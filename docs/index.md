# KiTverify SDK Documentation

## KiTverify SDK란?

KiTverify SDK는 음파 기반 앨범 태그 인식 기능을 제공하는 iOS SDK입니다.
마이크를 통해 음파 신호를 수신하고, 앨범 태그를 인식하여 인증된 결과를 반환합니다.

## 주요 기능

- **앨범 태그 인식**: 음파 신호를 통한 앨범 태그 자동 인식
- **인증 결과 반환**: 인식 성공 시 토큰 및 앨범 ID 제공
- **간편한 연동**: 초기화, 시작/종료의 단순한 API 구조
- **델리게이트 기반 이벤트 처리**: 성공, 실패, 종료 이벤트 콜백
